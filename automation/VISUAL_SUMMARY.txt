🆘 QUICK TROUBLESHOOTING
═════════════════════════════════════════════════════════════════════════════

"ImportError: No module named 'diffusers'"
$ pip install diffusers transformers accelerate

"FFmpeg not found"
macOS: brew install ffmpeg
Linux: sudo apt-get install ffmpeg
Windows: choco install ffmpeg

"API key invalid"
→ Regenerate from https://ai.google.dev

"CUDA out of memory"
→ Use CPU or reduce num_frames parameter

"Rate limited"
→ Wait 1 minute (60 requests/minute is the limit)

See COMPLETE_GUIDE.md for more troubleshooting.
