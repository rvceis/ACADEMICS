MATCHING ALGORITHM SUMMARY
==========================

The AI matching system finds optimal energy sellers for buyers by:

1. Taking buyer requirements (kWh needed, max price, preferences)
2. Querying available sellers from database
3. Calculating 6 individual scores for each seller
4. Applying weights to create combined score (0-100)
5. Sorting by score and returning top matches
6. Displaying all 6 scoring variables to user for transparency
7. User selects preferred matches
8. System creates smart_allocations linking buyer-seller pairs
9. Energy trading begins based on allocation

The system is designed to be:
- TRANSPARENT: Shows all scoring factors
- FLEXIBLE: Weights can be adjusted by admin
- EFFICIENT: Caches results for 5 minutes
- SCALABLE: Can handle 1000+ devices
- RELIABLE: Separate MQTT microservice

KEY DIFFERENTIATOR: Unlike simple distance-based matching,
this algorithm considers 6 factors including price, reliability,
renewable preference, and timing - making it a true "smart" allocation
system rather than just geographic proximity matching.
