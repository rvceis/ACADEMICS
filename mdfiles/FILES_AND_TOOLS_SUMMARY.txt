BACKEND EXPORT ENDPOINTS:
================================================================================

All endpoints require authentication (JWT token)
Available to all authenticated users (no role restrictions)

GET /schemas/export/json
  - Export all schemas with full metadata as JSON
  - Returns: List of schema definitions

GET /schemas/export/sql
  - Export all schemas as PostgreSQL CREATE TABLE statements
  - Returns: SQL script text

GET /schemas/{schema_id}/export/json
  - Export single schema as JSON
  - Returns: Schema definition

GET /schemas/{schema_id}/export/sql
  - Export single schema as SQL
  - Returns: PostgreSQL CREATE TABLE statement

GET /schemas/{schema_id}/export/download/{format}
  - Download schema export as file (JSON or SQL)
  - Formats: json, sql
  - Returns: File download

TROUBLESHOOTING:
================================================================================

Authentication Failed:
  - Ensure Flask backend is running on port 5000
  - Check database connection
  - Verify user exists and password is correct

Page Not Rendering:
  - Clear browser cache and reload
  - Check DevTools console for errors
  - Verify React frontend is running on port 3000

Export Returns 404:
  - Verify endpoints are in /flask_backend/app/routes/schemas_dynamic.py
  - Restart Flask backend after changes
  - Check schema exists and has proper ID

Data Validation Fails:
  - Use data_validator.py to check data quality
  - Review error messages for type mismatches
  - Ensure CSV has proper headers

Permission Denied (403):
  - Check if user is schema creator
  - Verify user role (Admin can delete all)
  - Check JWT token expiration
